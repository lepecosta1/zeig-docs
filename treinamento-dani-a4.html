Onde clicar em cada marketplace

MERCADO LIVRE
  1. Vendas → Etiquetas / A enviar
  2. Selecionar todos → Imprimir etiquetas
  3. Flex primeiro, em pilha separada
TIKTOK SHOP
  1. Pedidos → A enviar (To ship)
  2. Imprimir etiqueta (+ romaneio)
  3. Marcar como Enviado
SHOPEE
  1. Meus Pedidos → A enviar
  2. Imprimir etiqueta + lista de coleta
  3. Organizar envio

Impressão (Zebra)

  1. Impressora Zebra · papel 100×150 mm
  2. Margens = Nenhuma · Escala = 100%
  3. 1 etiqueta por página · código legível

Conferir sempre (≈1 min por pacote)

  • Referência (SKU) da etiqueta = peça
  • Tamanho certo
  • Cor certa
  • Peça sem defeito (costura, botão, mancha)

Antes de sair pra coleta

  • Flex pronto antes das 11:00
  • Etiqueta certa no pacote certo
  • Lista de coleta impressa (Shopee/TikTok)
  • Nº de pacotes = nº de pedidos
REGRA DE OURO: imprime tudo → separa → confere → embala. Nunca misture as etapas. Fazer o grosso de manhã libera a tarde e garante o Flex no mesmo dia.
